Module: TexturedSurfacePolygon

Constructs Textured Surface Polygon. The polygon is drawn using the standard attributes. Unlike the standard surface polygon it draws associated image on the area of the polygon.
Parameters:
Name Type Description
boundaries Array.<Location> Array of locations enclosing the surface polygon.
