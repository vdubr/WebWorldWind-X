About

This repository contains the extension to the Web World Wind library. It focuses on the UI elements and utilities that doesn't have place in the official repository as well as on the support to the less used formats that shouldn't grow the size of the core framework.

Examples

The full code for the examples below is available in the examples module of the Web WorldWind Extension project.

CyclicPickExample

Demonstrates different controller for picking. This controller selects all renderables on the given point. Remembers the selected one and if the user clicks on the same place renderable above the one will be selected. If there is none above then, the one below will be selected.

Source:
CyclicPickExample.js
CyclicPickExample.html

LayerOrderExample

Demonstrates utility class for automatic reordering of the layers. The default behavior is that the layers are ordered based on the position in the array. This allows ordering of the layers based on the z-index. The higher z-index means the layer is higher in the order.

Source:
LayerOrderExample.js
LayerOrderExample.html

TexturedSurfacePolygonExample

The standard surface polygon implementation doesn't allow texture to be used as the coverage. This implementation takes a texture and draws it as the cover of the polygon. It supports only the rectangular textures.
